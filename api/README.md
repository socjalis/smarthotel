## Project setup

```bash
$ docker-compose -p smarthotel up -d
```

```bash
./api $ yarn install
```

## Compile and run the project

```bash
# development
./api $ yarn run start

# watch mode
./api $ yarn run start:dev

# production mode
./api $ yarn run start:prod
```

## Build image

```bash
docker build -t smarthotel .
```
